Welcome to Checkers!

Please read the rules below before proceding

For this setup of checkers, each player has 6 pieces

  • The first player to move is black
  • Each player is only allowed to move on dark tiles diagonally.
  • If a jump is availble, it must be taken
  • Jumps can only be made diagonally when another empty square is available behind the opponent
  • Multiple jumps are not allowed
  • Backward movement is not allowed
  • Once all of one players pieces have been captured, the game is over
